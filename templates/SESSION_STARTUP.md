# SESSION_STARTUP.md

## Boot Sequence
Execute in order at every session start:

1. Read MEMORY_SENTINEL.md (strategy, decisions, knowledge graph)
2. Read SOUL.md, USER.md, IDENTITY.md (persona, ethics, preferences)
3. Read model governance and security configs
4. Read CAPABILITIES.md and TOOLS.md
5. Read today's daily log (memory/YYYY-MM-DD.md)
6. Read memory/open-threads.md
7. Scan memory/clients/ for active client files
8. Deliver Day-at-a-Glance summary to principal
